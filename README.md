# 📌 배경 & 문제 정의

제조 현장에는 4가지 구조적 문제가 존재합니다.

| 문제 | 설명 |
|------|------|
| 매뉴얼 탐색 어려움 | 설비 매뉴얼은 수백 페이지 분량이며 설비·버전별로 분산되어 있음 |
| 신입·협력사 교육 부담 | 도메인 용어 이해까지 수개월 교육 필요, 숙련자 의존도 높음 |
| 노하우 단절 | 현장 노하우가 개인·비공식 채널에만 존재해 조직 지식화 불가 |
| 불량·고장 대응 지연 | 유사 과거 사례가 문서·개인 경험에 흩어져 신속 대응 어려움 |

Manupilot은 이 네 가지 문제를 LLM과 RAG 기반으로 해결하는 **제조업 사내 업무 효율화 서비스**입니다.
